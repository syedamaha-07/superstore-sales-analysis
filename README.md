# Superstore Sales Analysis Project

## Project Overview
This project analyzes sales data using Google Sheets Pivot Tables and Charts.

## Tools Used
- Google Sheets
- Pivot Tables
- Data Visualization

## Analysis Performed

### 1. Monthly Sales Analysis
Sales were analyzed month wise to identify trends and seasonal performance.

Insight:
Sales fluctuate across months and peak in high demand periods.

---

### 2. Category Wise Sales
Technology and Furniture categories generate major revenue compared to others.

---

### 3. Regional Performance
Different regions show variation in sales performance.

---

## Conclusion
Sales trends help businesses understand customer demand and improve decision making.
